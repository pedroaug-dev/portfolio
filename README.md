# Portfólio - Pedro Augusto

Um portfólio pessoal moderno e responsivo desenvolvido com **HTML**, **CSS** e design focado em apresentar projetos, especialidades e informações profissionais.

## Sobre o Projeto

Portfólio web pessoal de Pedro Augusto, desenvolvedor em formação focado em:

- Web Development
- Automação
- Análise de Dados

O site foi desenvolvido com foco em performance, responsividade e experiência do usuário, apresentando um design moderno com tema escuro.

## Tecnologias Utilizadas

- **HTML5** - Estrutura semântica
- **CSS3** - Estilização e responsividade
- **Google Fonts** - Tipografia (Poppins)
- **Design Responsivo** - Mobile-first approach

## Estrutura do Projeto

```
portfolio/
├── index.html          # Arquivo principal HTML
├── assets/
│   └── style.css       # Folha de estilos
├── images/
│   ├── logo.png        # Logo do portfólio
│   └── pessoa.png      # Foto do desenvolvedor
└── README.md           # Este arquivo
```

## Seções do Portfólio

- **Início** - Apresentação pessoal e call-to-action
- **Especialidades** - Áreas de atuação e competências
- **Sobre** - Informações profissionais detalhadas
- **Projetos** - Portfólio de trabalhos desenvolvidos
- **Contato** - Opções para entrar em contato

## Como Usar

1. **Clone o repositório:**

```bash
git clone https://github.com/seu-usuario/portfolio.git
cd portfolio
```

2. **Abra o arquivo `index.html` no navegador:**
   - Clique duplo no arquivo `index.html`, ou
   - Use uma extensão como Live Server no VS Code

3. **Personalize o conteúdo:**
   - Edite as informações no `index.html`
   - Atualize os estilos no `assets/style.css`
   - Substitua as imagens em `images/`
